在2020年4月4日如何...

初次写于 2020-04-04
这篇文章最后修改于 2020-04-04 ，本页内容可能已经过时！

2020年4月4日本来是个清明节，扫墓的日子，结果因为疫情，很多人牺牲了。于是呢，国家便在这个日子开展国家悼念活动。

手机（Android）

因为我没有苹果手机，就不提供它的教程了，反正大致步骤跟安卓瘟10一致

如上图。打开设置》…(据手机而不同)…》开发者模式》模拟颜色空间，选择全色盲

好像安卓9以后有个什么深色模式，配合那个应该体验更佳。

电脑（Windows10）

如上图。打开设置》轻松使用》颜色滤镜，把它打开，下面选择灰度。

旧版Windows10应该是类似的操作。

还可以开个深色模式。

网页（Html+CSS）

嘿嘿，这才是真正的技术性内容

网页黑白非常简单，加个CSS就行。

html {
    filter: grayscale(100%);
    -webkit-filter: grayscale(100%);
    -moz-filter: grayscale(100%);
    -ms-filter: grayscale(100%);
    -o-filter: grayscale(100%);
    filter: progid:DXImageTransform.Microsoft.BasicImage(grayscale=1);
    -webkit-filter: grayscale(1);
}

实测加在html和body都行。
